# Proposta de redesign do Meu SUS Digital 

Este projeto faz parte do meu TCC, onde avalio a acessibilidade do Meu SUS Digital sob os requisitos e recomendações da ABNT NBR 17060 e a partir disso, faço um protótipo com o intuito de melhorar os problemas de acessibilidade encontrados.

O projeto foi desenvolvido usando HTML, CSS e JS, juntamente utilizando os componentes do [Design System do governo](https://www.gov.br/ds/home).

Ao ser clonar o repositório, é necessário que seja feito o comando `npm install` para instalar as dependências do projeto na pasta *node_modules*.

O projeto também está disponível no GitHub Pages a partir deste [link](https://mariafcknt.github.io/projeto-meu-sus-digital/).
